# sf

sfは[[Simple Feature Access]]にもとづいて，[[地理空間データ]]を扱うための[[Rパッケージ]]，またはそのパッケージにより[[フィーチャ]]を扱うためのクラスを指す[^web-sf]．[[ジオメトリ]]の長さや面積を計測するための関数や，[[ポリゴン]]を条件にもとづいて分割する関数など，ジオメトリを操作するための関数を提供する．また，地理空間データを表現するためのクラスを提供している．具体的には，単体のジオメトリ，複数のジオメトリ，フィーチャのそれぞれを扱うクラスとして，[[sfg]]（simple fature geometry），[[sfc]]（simple feature list-column），[[sf]]（simple feature）を提供する．

[^web-sf]: Edzer Pebesma, Simple Features for R • sf, URL: https://r-spatial.github.io/sf/
