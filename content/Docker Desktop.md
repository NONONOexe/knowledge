---
title: Docker Desktop
---

Docker Desktopとは，Windows，Mac，またはLinuxといったデスクトップ環境で，[[Docker]]を利用するためのアプリケーションである[^web-docker_desktop]．このアプリケーションには，[[Docker Engine]]，[[Docker Compose]]など，Docker環境に必要なツールがまとめられている．Docker Desktopをインストールすることで，これらのツールを個別にインストール，設定する手間を省き，手軽にDocker環境を構築できる．また，[[コンテナ]]や[[イメージ]]の管理，アプリケーションの実行状況の監視などをおこなうことができるGUIを提供する．

[^web-docker_desktop]: Docker Docs, Docker Desktop, URL: https://docs.docker.com/desktop/
