# Edit

Editは[[Microsoft]]が提供するエディタである．[[MS-DOS Editor]]に敬意を表しつつも，[[VS Code]]のようにモダンなインターフェースとコントロールを備える[^web-microsoft_edit]．2025年5月19日にバージョン1.0.0がリリースされた．[[オープンソース]]であり，[[Rust]]で開発されている．また，[[winget]]にも対応しており，以下のコマンドからインストールできる．

```powershell
winget install Microsoft.Edit
```

[^web-microsoft_edit]: Microsoft, microsoft/edit (GitHub), URL: https://github.com/microsoft/edit
