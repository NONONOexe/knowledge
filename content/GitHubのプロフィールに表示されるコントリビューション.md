---
title: GitHubのプロフィールに表示されるコントリビューション
---

[[GitHub]]のプロフィールには[[コントリビューション]]が表示される．これには，次のものが含まれる[^web-why_my_contribution]．

- 新しいリポジトリの作成
- 既存のリポジトリのフォーク
- ブランチへのコミット
- Issueのオープン
- Discussionのオープンまたは回答
- プルリクエストの提案
- プルリクエストレビューの送信

表示されるコミットには，条件がある．次のすべての条件を満たすコミットのみがプロフィールに表示される．

- コミットに使用されたメールアドレスが，GitHubのものと同じであること
- コミットがフォークでなく，スタンドアロンのリポジトリでおこなわれたものであること
- コミットがデフォルトブランチ，または`gh-pages`ブランチでおこなわれたものであること

[^web-why_my_contribution]: GitHub Docs, “コントリビューションがプロフィールに表示されないのはなぜですか？,” URL: https://docs.github.com/ja/account-and-profile/setting-up-and-managing-your-github-profile/managing-contribution-settings-on-your-profile/why-are-my-contributions-not-showing-up-on-my-profile
