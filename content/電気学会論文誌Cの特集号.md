| 投稿締切月 | 特集分野              |

|    4月 | システム技術            |
|    5月 | 電子デバイス 制御技術       |
|    6月 | 光・量子デバイス          |
|    7月 | 医用生体              |
|    8月 | 通信技術<br>情報処理      |
|    9月 | 電子・情報・システム部門大会    |
|   10月 | 電子材料<br>情報システム    |
|   11月 | 知覚情報<br>国際会議ICESS |
|   12月 | 四国支部連合大会          |
|    1月 | 関西支部連合大会          |
|    2月 | 東海支部連合大会          |
|    3月 | 電子回路              |

[^web-iee_c]: 一般社団法人 電気学会，C部門誌 特集号の論文募集，URL: https://www.iee.jp/eiss/journal/special
